## 🚀 Featured Projects

### 📦 [newspaper-tracker](#)
A GPS-based Android app designed for tracking newspaper delivery routes. 
* **Features:** Auto street-switching, tap-to-deliver, long-press-to-flag, and run history with personal bests.
* **Tech Stack:** Kotlin, Room, FusedLocationProvider, OSMDroid.

### 🔭 [openscan-pi](#)
A DIY photogrammetry and IR 3D scanner built on the OpenScan platform. Budget build, serious results.
* **Hardware:** Raspberry Pi, stepper motors, VL53L5CX/L8CX depth sensors, OV9281 global shutter camera.

### 🐧 [rocm-setup-notes](#)
Battle-tested documentation for running DaVinci Resolve on AMD + Arch Linux. Because someone had to document this.
* **Covers:** ROCm 7.x, mesa-git conflicts, Intel OneAPI TBB interference, and ABI issues.

An interactive, multi-page space education website tailored for students and educators. 
* **Features:** Single self-contained HTML file, glassmorphism styling, and a purple/blue theme toggle.
